Session 7 — HTML5 APIs

Local Storage & Session Storage

localStorage saves data in the browser with no expiration date — it persists even after closing the tab. sessionStorage works the same way but clears when the browser tab is closed. Both store simple key-value pairs.


Geolocation API

The Geolocation API asks the browser to share the device's coordinates — the browser will prompt for permission first.

Canvas: Drawing Simple Shapes

The <canvas> element gives JavaScript a drawable surface for shapes, graphs, or games.
