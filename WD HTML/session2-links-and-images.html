Session 2 — Working with Links and Images

Anchor Tags: <a href="…">

Links are created with the anchor tag and a href attribute:

  • External link (absolute path): Visit Google
  • External link (opens in new tab with target="_blank"): MDN Web Docs
  • Internal link (relative path, would point to a page in the same folder): About Page (relative path example)
  • Email link: Send an Email

Image Tags: <img src="…">

Images are embedded with the <img> tag. It's a self-closing tag that requires a src (source) and should always include an alt attribute for accessibility.

Alt text describes the image for screen readers and displays if the image fails to load. It's also important for SEO, since search engines read it to understand image content.

Relative vs Absolute Paths

Type Example Meaning
Absolute https://example.com/img.jpg Full URL — works from anywhere
Relative images/photo.jpg Path relative to the current file's location

Why Alt Text Matters for Accessibility & SEO

Screen readers announce alt text aloud so visually impaired users know what an image shows. Search engines also use alt text to index images, which can drive additional traffic through image search results. An image without alt text is both inaccessible and a missed SEO opportunity.
